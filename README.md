# IPL Runs Prediction

This project uses **Linear Regression** to predict IPL player runs based on:

* Matches
* Average (AVG)
* Strike Rate (SR)
* Fours (4s)
* Sixes (6s)

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

## Model

Linear Regression

## Evaluation

* MAE
* MSE
* R² Score

## Visualizations

* Feature Importance Plot
* Residual Plot
* Actual vs Predicted Plot
